== Description ==

This plugin will allow you to place a block with reviews on any page. Users of the site have the opportunity to choose to post their nickname in the review or edit it before publishing the review. It is also possible to choose whether to show the avatar or not. They will be able to put ratings (stars). To insert to the page, use the shortcode [ms_reviews]
